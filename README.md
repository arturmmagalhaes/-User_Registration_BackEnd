## Rodar o projeto
```
  git clone https://github.com/arturmmagalhaes/User_Registration_BackEnd.git
  cd User_Registration_BackEnd
  npm install
  touch .env
  npm run migration
  npm start
```
- .env
```
  DB_NAME = 
  DB_USERNAME = 
  DB_PASSWORD = 
  DB_HOST = 

  JWT_KEY = 
```

#### Rodar tests
```
  git clone https://github.com/arturmmagalhaes/User_Registration_BackEnd.git
  cd User_Registration_BackEnd
  npm install
  npm run test
```
